# @fiap-blog/eslint-config

Shared ESLint configurations for FIAP Blog monorepo.

## Usage

### React

```js
// eslint.config.js
module.exports = {
  extends: ['@fiap-blog/eslint-config/react']
};
```

### Node.js

```js
// eslint.config.js
module.exports = {
  extends: ['@fiap-blog/eslint-config/node']
};
```
